1.  **Open Command Prompt or PowerShell:**
    *   Search for "cmd" or "PowerShell" in the Windows search bar and open the application

2.  **Run the command:**
    *   In the terminal window, type the following command and press Enter:
        ```bash
        java -version
        ```
